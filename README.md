# The Pouma Academy

Next.js website for The Pouma Academy — programs, contact, student resources, and admin panel.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Admin: [http://localhost:3000/admin](http://localhost:3000/admin)  
Local dev password (only if `ADMIN_PASSWORD` is unset): `dev-only-change-me`  
Email: `ask@thepoumaacademy.com`

Copy `.env.example` to `.env.local` to override settings locally.
